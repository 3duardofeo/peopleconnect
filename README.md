# People Connect App

A React application for discovering and connecting with people from different countries and cultures, featuring an authentic old Apple Mac design aesthetic.

## Features

- **Cultural Diversity**: Browse people from 15+ different countries and cultures
- **Advanced Search**: Search by name, city, country, or zip code
- **Location-Based**: See distance indicators to find people near you
- **Responsive Design**: Fully responsive across all devices
- **Retro Mac UI**: Authentic old Apple Macintosh design system
- **Rich Profiles**: Detailed profiles with interests, languages, and cultural information

## Technologies Used

- React 18
- Vite
- Lucide React (icons)
- CSS3 with retro Mac styling
- Responsive Grid Layout

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Design Features

- **Classic Mac UI Elements**: Title bar with window controls, menu bar, and classic button styles
- **Pixel-Perfect Styling**: Authentic 1980s Macintosh visual design
- **Responsive Grid**: Adaptive card layout that works on all screen sizes
- **Color-Coded Distance**: Visual distance indicators using classic Mac colors
- **Retro Typography**: Chicago font family for authentic Mac feel

## Sample Data

The app includes 15 sample profiles representing diverse cultures including:
- Japanese, Mexican, Egyptian, Indian, Danish
- Nigerian, Chinese, Brazilian, Emirati, French
- Ghanaian, Russian, Saudi Arabian, Korean, Moroccan

Each profile includes cultural information, languages, interests, and location data.
